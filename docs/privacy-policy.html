Privacy Policy

Last updated: January 2025

TL;DR: We don't collect, store, or transmit any personal data or browsing information. Everything happens locally on your device.

Overview

Patreon Exporter is a browser extension that helps users export Patreon posts to clean PDF files for offline reading. This privacy policy explains our data practices and your privacy rights.

Data Collection

We do not collect, store, or transmit any personal data or browsing information.

The extension operates entirely locally on your device and does not send any data to external servers, analytics services, or third parties.

Local Storage Only

The extension stores only user preferences locally on your device using your browser's built-in storage:

  • PDF image quality settings (high/medium/low)
  • Comment inclusion preferences (whether to include comments in PDFs)
  • Download dialog preferences (whether to show save dialog)

This data:

  • Never leaves your device
  • Is not shared with any third parties
  • Is not accessible to us or anyone else
  • Can be cleared by uninstalling the extension or clearing browser data

Permissions Explained

The extension requests these permissions for its core functionality:

activeTab

Allows the extension to access the current Patreon page content to extract post text and images for PDF generation. We only access the currently active tab when you explicitly use the extension.

downloads

Enables the extension to save the generated PDF files to your device's download folder using your browser's download system.

storage

Stores your preferences locally in your browser so they persist between sessions. No external storage is used.

Host Permissions (patreon.com)

Allows access to Patreon domains to read the post content you want to export. This permission is limited to Patreon domains only.

Data Processing

When you use the extension, here's exactly what happens:

  1. It reads only the current Patreon post you're viewing
  2. Processes the content locally in your browser to remove navigation, ads, and clutter
  3. Optionally expands and includes comments if you've enabled this feature
  4. Converts images to prevent external loading issues in the PDF
  5. Generates a PDF file entirely on your device using JavaScript libraries
  6. Saves the PDF using your browser's built-in download system

No data is sent to external servers at any point in this process. All PDF generation happens locally in your browser using client-side JavaScript.

Third-Party Services

This extension does not integrate with or send data to any:

  • Analytics platforms (Google Analytics, etc.)
  • External APIs or web services
  • Social media platforms
  • Advertising networks
  • Data collection services

Image Handling

When processing Patreon posts that contain images:

  • Images are fetched directly by your browser using standard web protocols
  • Images are converted to base64 format locally for inclusion in the PDF
  • No image data is sent to external processing services
  • If external images can't be loaded due to CORS restrictions, they're replaced with placeholders

Browser Compatibility

This privacy policy applies to the extension when used on:

  • Google Chrome and Chromium-based browsers
  • Mozilla Firefox
  • Microsoft Edge
  • Other browsers that support WebExtensions

Data Security

Since all processing happens locally on your device:

  • Your data never travels over the internet
  • There are no external servers that could be compromised
  • Your browsing patterns and exported content remain private
  • The extension cannot access data from other websites or tabs

Children's Privacy

This extension does not knowingly collect any information from children under 13 years of age. Since we don't collect any personal information at all, this extension is safe for users of all ages.

Changes to This Policy

If we make changes to this privacy policy, we will:

  • Update the "Last updated" date above
  • Notify users through the extension's update mechanism if changes are significant
  • Maintain the same privacy-first principles

Continued use of the extension after changes constitutes acceptance of the revised policy.

Your Rights

Since we don't collect any personal data, traditional data protection rights like data deletion or portability don't apply. However, you can:

  • Uninstall the extension at any time
  • Clear your browser's extension data to reset all preferences
  • Review the extension's source code on GitHub
  • Contact us with questions about our privacy practices

Contact Information

For questions about this privacy policy or the extension, you can:

  • Create an issue on our GitHub repository
  • Contact us through the Chrome Web Store support page
  • Email us at: privacy@yourextension.com (if you have a contact email)

Legal Compliance

This extension and privacy policy comply with:

  • Chrome Web Store Developer Program Policies
  • Mozilla Add-on Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
  • Other applicable privacy laws and regulations

Open Source

This extension is open source, which means:

  • You can review the complete source code on GitHub
  • Security researchers can audit our privacy practices
  • The community can contribute to improvements
  • You can verify that we're doing exactly what we say we're doing

Transparency: The best privacy policy is code you can read. Our complete source code is available on GitHub for your review.

This privacy policy applies to the Patreon Exporter browser extension available on the Chrome Web Store, Firefox Add-ons, and other browser extension stores.
